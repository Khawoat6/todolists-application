### Set up

1. Clone or download this repo
    ``` 
        git clone https://github.com/Khawoat6/todolists-application.git 
    ```
2. open project 
    ```
        cd todolists-application
    ```
3. Install required packages 
   ```bash
        npm install
   ```
4. **Initialize** all projects
   ```bash
       npm start
   ```
